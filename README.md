# Desmos 
A fun way to learn and visualize math. The goal was to understand the shapes of trigonometric functions (sin, cos, cosh, tan, tanh) and hyperbolic functions. [Link to desmos](https://www.desmos.com/calculator/a35fc4765a)


<img width="853" height="590" alt="image" src="https://github.com/user-attachments/assets/c660561e-75f2-4bd4-8423-12b657d7ae81" />


Intially the image is sketch out on a notebook. I break down the drawing into different shapes and lines, then try and match the shape of each equation to the image, with restricted domain and range. The equations are organised into parts like nose, eyes, shading etc. The total number of equations is 387.
